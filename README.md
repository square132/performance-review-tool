# performance-review-tool
Performance review and development tracking tool 
# The problem:
In scaling organizations, people management is often playing catch-up. At annual review time, without clear guidance, people managers run the risk of conflating performance attainment with career advancement. This develops an organization expectation with employees that promotion cycles are annual, or confusion on logic behind raises. 

# The objective:
This tool analyzes weekly and quarterly 1:1 meeting notes, distinguishes performance attainment against job description or running priorities for compensation considerations, and outlines progress against career advancement and pathing goals. Because the analysis is AI generated against meeting notes that both a report and manager review, the analysis is designed for objectivity and clarity.

# Notes:
* For those who are not using tools like Lattice or others for their 1:1's and quarterly development meetings with their team members, this app is an upgrade for start-up colleagues who are using tools like g-docs, OneNote, or trello to track performance and development.

* This is designed to be a proto-type. To encrypt and secure the data, include role based permissions, there is more set up. Or, you can just buy/use more feature extensions within people management tools.
